# VBYTDownloader v1.0

A VB.NET YouTube downloader powered by yt-dlp and FFmpeg.

![VBYTDownloader Screenshot] Screenshot 2026-06-03 121832.png

FEATURES

* MP4 Downloads (360p, 480p, 720p, 1080p)
* MP3 Downloads
* Playlist Downloads
* Thumbnail Preview
* FPS Detection
* Codec Detection
* Estimated Video Size
* Estimated Playlist Size
* Download History
* Dark Mode
* Update Checker

HOW TO USE

1. Paste a YouTube video or playlist URL.
2. Select MP4 or MP3.
3. Choose quality.
4. Click Download.
